Получение собственных API ключей

Если вы хотите использовать собственные API ключи Telegram:

  1. Перейдите на сайт my.telegram.org/apps
  2. Войдите в свой аккаунт Telegram
  3. Создайте новое приложение
  4. Скопируйте API ID и API Hash
  5. Вставьте их в соответствующие поля формы
Внимание! Храните ваши API ключи в безопасности.
Файлы сессий

Файлы сессий хранят информацию об авторизации в Telegram. Если у вас возникают проблемы с авторизацией, вы можете изменить имя сессии для создания новой.

Совет: Для каждого номера телефона рекомендуется использовать отдельную сессию.
